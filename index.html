Diagnostic Testing Tool

This is an attempt to visualise test characteristics such as sensitivity, specificity, PPV, NPV. Hopefully it will enlighten rather than confuse

A fixed population of 10,000 is used. the sliders are used to adjust disease prevalance, sensitivity and specificity - which results in a confusion matrix, test calculations and some graphs. A particular use case is to see how disease prevelance affects positive and negative predictive values
